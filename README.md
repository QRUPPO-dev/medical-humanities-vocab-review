# 医学人文英语复习小程序 macOS 发布包

这个分支只保存 macOS 本地版可下载发布物。源码在 `main` 分支。

## 下载

- macOS 本地版：`packages/medical-humanities-english-review-macos-v1.0.0.zip`

压缩包包含打包后的网页程序、macOS 本地静态服务和一键启动脚本。解压后双击 `启动mac版.command` 或 `start-mac.command` 即可运行。

## 分支边界

- `main`：React/Vite/Capacitor 源码和项目配置。
- `release`：macOS 本地版 zip、发布说明和校验和。

## 校验

SHA-256 校验和见 `CHECKSUMS.txt`。
