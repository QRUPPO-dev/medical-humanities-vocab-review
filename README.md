# 医学人文英语复习小程序发布包

这个分支只保存可下载发布物。源码在 `main` 分支。

## 下载

- Windows 本地版：`packages/medical-humanities-english-review-windows-v1.0.0.zip`
- macOS 本地版：`packages/medical-humanities-english-review-macos-v1.0.0.zip`

两个压缩包都包含本地静态网页程序和一键启动脚本。解压后按包内说明运行即可。

## 分支边界

- `main`：React/Vite/Capacitor 源码和项目配置。
- `release`：打包后的 Windows/macOS 本地版 zip、发布说明和校验和。

## 校验

SHA-256 校验和见 `CHECKSUMS.txt`。
