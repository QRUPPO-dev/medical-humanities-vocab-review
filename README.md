# 医学人文英语复习小程序

这是一个用于复习复旦大学《医学人文英语》课程相关课本练习题与知识点的小程序。当前 GitHub 仓库提供源码，并在 `release` 分支和 GitHub Releases 中提供 Windows、macOS 本地版压缩包，下载后解压即可在自己的电脑上运行。

本项目是非官方学习辅助工具，主要用于课程复习、课堂抽查准备和个人记忆巩固。

题目内容均整理自公开可获得的《医学人文英语》课本练习题。本项目由 vibe coding 方式制作，主要目标是帮助同学快速抽查和复习；不保证所有中文释义、填词答案、翻译答案都完全准确。复习时请以课程教材、课堂讲解和任课教师要求为准。

## 功能

- 按单元或总体范围抽查复习
- 支持单词、词组、填空词、翻译等题型
- 自动记录练习进度、正确率和薄弱项
- 可把不熟的内容加入生词本
- 支持单独练习生词本
- 支持导出和导入进度，方便换电脑继续复习

## Windows 使用方法

1. 在 GitHub Releases 页面下载 Windows 版压缩包。
2. 右键压缩包，选择“全部解压”或“解压到当前文件夹”。
3. 进入解压后的文件夹。
4. 双击 `启动Windows版.bat`，也可以双击英文别名 `start-windows.bat`。
5. 浏览器会自动打开 `http://127.0.0.1:5179/?v=windows`。
6. 使用期间不要关闭弹出的本地服务窗口；关闭后网页也会无法访问。

注意：不要直接双击 `app/index.html`。部分浏览器会因为本地文件限制导致页面空白或功能异常。

## macOS 使用方法

1. 在 GitHub Releases 页面下载 macOS 版压缩包。
2. 双击压缩包解压。
3. 进入解压后的文件夹。
4. 双击 `启动mac版.command`，也可以双击英文别名 `start-mac.command`。
5. 浏览器会自动打开类似 `http://127.0.0.1:5179/?v=macos` 的地址。
6. 使用期间不要关闭弹出的 Terminal 窗口；关闭后网页也会无法访问。

注意：不要直接双击 `app/index.html`。部分浏览器会因为本地文件限制导致页面空白或功能异常。

## 为什么别人打不开我发的 127.0.0.1 链接？

`127.0.0.1` 表示“当前这台电脑自己”，不是互联网地址。

所以你把 `http://127.0.0.1:5179/` 发给别人时，对方访问的是他自己电脑上的本地服务。如果他没有下载项目并启动本地版，浏览器就会显示“拒绝连接”。

正确分享方式是：把 GitHub Releases 里的压缩包链接发给对方，让对方下载、解压后在自己的电脑上运行。

## 常见问题

### 显示 127.0.0.1 拒绝连接

通常是本地服务没有启动。

请确认：

- 已经解压整个文件夹，不是在压缩包里面直接运行
- `app` 文件夹和启动脚本在同一个目录
- 已经双击对应系统的启动脚本
- 弹出的本地服务窗口没有被关闭
- 如果之前开过旧窗口，先全部关闭，再重新双击启动

### Windows 提示不安全或阻止运行

这是因为 `.bat` 和 `.ps1` 是本地启动脚本。它们只会在本机启动一个静态网页服务，用来打开当前文件夹里的网页文件。

如果 Windows SmartScreen 阻止运行，可以选择“更多信息”后继续运行。请只从可信来源下载本项目压缩包。

### macOS 提示没有权限或来自未知开发者

如果双击后提示文件没有执行权限，请打开“终端”，进入解压后的文件夹，执行：

```sh
chmod +x start-mac.command
```

然后再双击 `start-mac.command`。

如果 macOS 提示来自未知开发者，可以右键点击启动脚本，选择“打开”。

### 端口 5179 被占用

Windows 版如果提示端口被占用，可以先关闭之前打开的“Medical English Local Server”窗口，再重新运行。

仍然不行的话，可以编辑 `启动Windows版.bat` 或 `start-windows.bat`，把：

```bat
set "PORT=5179"
```

改成：

```bat
set "PORT=5180"
```

然后重新双击启动。

macOS 版会自动尝试使用 `5179` 到 `5199` 之间的空闲端口。如果这些端口都被占用，请先关闭旧的 Terminal 服务窗口，再重新运行。

## 进度保存

练习进度保存在当前浏览器里。

如果要换电脑：

1. 在旧电脑点击“导出进度”。
2. 在新电脑打开本程序后点击“导入进度”。
3. 选择导出的 JSON 文件。

## 项目结构

`main` 分支为源码分支：

```text
.
├── src/                  # React 小程序源码和题库数据
├── public/               # 图标、manifest 和 service worker
├── android/              # Capacitor Android 工程
├── ios/                  # Capacitor iOS 工程
├── package.json          # 项目脚本和依赖
├── pnpm-lock.yaml        # pnpm 锁定文件
└── README.md             # GitHub 项目说明
```

`release` 分支为发布物分支：

```text
.
├── packages/             # Windows/macOS 本地版压缩包
├── CHECKSUMS.txt         # SHA-256 校验和
├── RELEASE_NOTES.md      # 版本说明
└── README.md             # 发布包说明
```
